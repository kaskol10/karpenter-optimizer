# Adopters

This page lists organizations and individuals using Karpenter Optimizer in production or for testing purposes. 

**Are you using Karpenter Optimizer?** We'd love to hear about it! [Submit a pull request](https://github.com/kaskol10/karpenter-optimizer/pulls) to add your organization to this list.

## How to Add Your Organization

1. Fork the repository
2. Add your organization to the list below (alphabetically)
3. Include:
   - Organization name (with optional logo)
   - Brief description of use case (optional)
   - Link to your organization's website (optional)
4. Submit a pull request
